# 🚀 Featured Projects

## 🔍 [Jignasa](https://github.com/Abiram116/Jignasa)

Local-first RAG document assistant with a ReAct tool-calling agent and four adaptive search modes.

**FastAPI • React/TypeScript • LangChain • FAISS • Ollama • Docling • RAGAS**

- Offline RAG pipeline powered by Ollama and FAISS
- ReAct agent supporting Chat, Knowledge, Web, and Hybrid search
- SSE streaming, persistent memory, response caching, prompt-injection guardrails, rate limiting, and installable as a PWA

---

## 📱 [SpaceLearn](https://github.com/Abiram116/SpaceLearn)

Cross-platform student productivity application with AI-powered learning tools.

**React Native • Supabase • LLM APIs**

- AI chat assistant, assignment generation, and learning companion
- Authentication and cloud storage using Supabase
- Built to explore production integration of LLM APIs in a mobile application
