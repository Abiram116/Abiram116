<h3>03 &nbsp;·&nbsp; PROJECTS</h3>

<sub>~/03-projects</sub>

<br/>

**01 / JIGNASA**
Local-first RAG document assistant. ReAct tool-calling agent with four adaptive search modes — chat, knowledge, web, hybrid. SSE streaming, persistent memory, response caching, rate limiting, prompt-injection guardrails, RAGAS-evaluated. Installable PWA.
<sub>FASTAPI · REACT · TYPESCRIPT · LANGCHAIN · FAISS · OLLAMA · DOCLING · SQLITE</sub>
<sub>[github.com/Abiram116/Jignasa](https://github.com/Abiram116/Jignasa)</sub>

<br/>

**02 / HANDWRITTEN DIGIT RECOGNITION**
CNN trained on MNIST wrapped in a Django interface. Draw a digit in-browser, get the prediction live from the served model.
<sub>PYTHON · DJANGO · TENSORFLOW · KERAS</sub>
<sub>[github.com/Abiram116/Handwritten-Digit-Recognition](https://github.com/Abiram116/Handwritten-Digit-Recognition)</sub>

<br/>

**03 / IMAGE COLORIZATION**
Deep-learning model for colorizing grayscale images. U-Net plus autoencoder architecture, CUDA-accelerated training pipeline.
<sub>PYTHON · TENSORFLOW · KERAS · OPENCV</sub>
<sub>[github.com/nandhithr6/Image-Colorization](https://github.com/nandhithr6/Image-Colorization)</sub>

<br/>

**04 / SPACELEARN**
React Native productivity app for students. AI chat, assignment generation, and a learning assistant driven by LLM APIs.
<sub>REACT NATIVE · EXPO · SUPABASE · LLM APIS</sub>
<sub>[github.com/Abiram116/SpaceLearn](https://github.com/Abiram116/SpaceLearn)</sub>

<br/>

**05 / LINSOLVERX**
Django web app that solves linear programming problems end to end — Simplex, Graphical, Transportation — with step-by-step working shown to the user.
<sub>PYTHON · DJANGO · NUMPY</sub>
<sub>[github.com/Abiram116/LinSolverX](https://github.com/Abiram116/LinSolverX)</sub>

<br/>

**06 / HACKATHON**
Longformer-based binary classifier fine-tuned for a long-document task. Hugging Face Transformers pipeline end to end.
<sub>PYTHON · PYTORCH · TRANSFORMERS · LONGFORMER</sub>
<sub>[github.com/nandhithr6/HackAthon](https://github.com/nandhithr6/HackAthon)</sub>

<br/>

**07 / RECIPEASY**
Full-stack recipe manager. Google OAuth, meal-planning calendar, animated interactions with GSAP.
<sub>PYTHON · DJANGO · POSTGRESQL · GSAP</sub>
